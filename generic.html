Warzone Stat Tracker

https://github.com/karenseunsom/warzone-project

Summary

This application was built for the purpose of easily tracking your Warzone stats in a fun way. You are given the option of picking which platform you'd like to view your stats on, such as Playstation, PC, XBOX, and more. On top of simply viewing stats, you are also given the capability of pinning users in order to compare side by side.

Technologies used

React, Styled Components, React-Bootstrap

Memory Game

https://github.com/karenseunsom/memory-game

Summary

A simple memory game to exercise your brain. A total of 16 cards and 2 cards with the same logo need to be chosen consecutively in order to remain up. Get all cards to be facing up in order to win the game. Mostly built with the intention of getting familiar with class components in React.

Technologies used

React, Bootstrap, CSS
